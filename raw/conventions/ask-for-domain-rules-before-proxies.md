When a task implies an ordering or grouping that depends on a domain concept not visible in the data, don't substitute a geometric or structural proxy and present the result as derived — ask for the actual rule, or for the reference point's location, before computing. A plausible-looking proxy metric can be confirmed wrong by the user after being applied; two clarifying questions up front would have caught this before any edits were made.
