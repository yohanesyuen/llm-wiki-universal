# Numbered Lists for Referenceable Items

Use numbered lists (`1.`, `2.`, ...), not bullets, for any list of discrete items the user might refer back to by index — findings, options, action items, tooling gaps. This lets the user say "do 2 and the last one" without a renumbering round-trip.
